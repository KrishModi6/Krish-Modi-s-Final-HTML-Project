Cookie Clicker Scratch Project

This project is a simple clicker game inspired by the popular Cookie Clicker game. The goal is to create a fun and engaging experience where players can click to earn points and unlock rewards. Here is the link: Cookie Clicker Scratch Project.

Inspiration and UI Design

I thought of a clicking game to create as I used to play a game called cookie clicker a lot. I want to implement a simple version of it on scratch.

I then start designing the game UI, focusing on the main clicking area and the incremental rewards system.

Code Implementation

I began by coding the basic mechanics of the game, including the click counter and reward system.

I implemented the click event that increments the score and updates the UI accordingly.

I added a simple reward system that grants bonuses for reaching certain milestones, enhancing the gameplay experience.

I then worked on the visual feedback for clicks, ensuring that each click feels satisfying and rewarding and added a grand prize in the end by getting the orange apple.

I implemented a final animation for the grand prize, making it a memorable moment for players.

I also added sound effects to enhance the user experience, making each click and reward feel more engaging.

I tested the game thoroughly, ensuring that all mechanics worked as intended and that the UI was responsive and intuitive. After server test sessions I made final adjustments to the UI and code, ensuring everything was polished and ready for release and officially completed this project.

Reflection

This project was a great learning experience, allowing me to apply my UI design skills and coding knowledge in a practical way. I enjoyed the process of creating a simple yet engaging game that can be expanded in the future.

I look forward to sharing this project with others and possibly expanding it with more features in the future.
